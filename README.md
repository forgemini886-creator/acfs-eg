# موقع الاستشراف الأنثروبولوجي لقضايا إفريقيا

هذا المستودع يستضيف موقعًا إلكترونيًا يعمل كمنصة استشرافية أنثروبولوجية تهتم بقضايا القارة الأفريقية. يجمع الموقع بين مناهج الأنثروبولوجيا الميدانية وأدوات الاستشراف الاجتماعي لتحليل تحولات المجتمعات، السياسات، والبيئات عبر أنحاء أفريقيا.

## الرؤية
أن نصنع مساحة معرفية متعددة التخصصات تُسهِم في فهم التحولات الاجتماعية والثقافية في أفريقيا وتقديم رؤى وتصورات مستقبلية مبنية على بيانات ميدانية، سرديات محلية، وتحليلات استشرافية.

## الرسالة
- توثيق وتفسير ديناميات التغيير الاجتماعي والثقافي عبر منظور أنثروبولوجي.
- إنتاج سيناريوهات واستشرافات سياسية/ثقافية لمساعدة الباحثين، صانعي السياسات، والمجتمعات المحلية على التفكير الاستراتيجي للمستقبل.
- نشر موارد قابلة لإعادة الاستخدام: تقارير، خرائط تفاعلية، مجموعات بيانات وصفية، ومقالات تحليلية.

## مواضيع رئيسية
- التحولات الحضرية والريفية
- الهجرة والنزوح
- التغير المناخي والموارد
- السياسات التنموية والحوكمة
- الذاكرة والهوية والممارسات الثقافية
- التكنولوجيا والمجتمع (رقمنة التراث، الاتصالات، شبكات المعرفة)
- الأمن الغذائي والاقتصاد المحلي

## أنواع المحتوى
- مقالات وتحليلات أنثروبولوجية واستشرافية
- دراسات حالة ميدانية ومقابلات موثقة
- خرائط ومرئيات جغرافية (GIS) وواجهات تحليلية
- مجموعات بيانات وصفية وروابط لمصادر أولية
- أدوات منهجية: أدلة ميدانية، نماذج سيناريو، طرق مشاركة المجتمعات

## المنهجية
نستخدم مزيجًا من:
- العمل الميداني الإثنوغرافي والمقابلات المتعمقة
- تحليل بيانات كمية ونوعية
- ورش استشرافية مع فاعلين محليين لصياغة سيناريوهات مستقبلية
- تعاون عبر شبكات بحثية ومجتمعية لضمان تعددية الأصوات والسياقات

## للجمهور
الموقع موجّه إلى:
- باحثين وطلاب الأنثروبولوجيا والدراسات الأفريقية
- صانعي سياسات ومخططي تنمية
- منظمات مجتمع مدني ومبادرات محلية
- أي مهتم بفهم مستقبل القارة من منظور إنساني-اجتماعي

## كيف تساهم
- فتح Issue لمناقشة فكرة أو مادة تريد نشرها
- إرسال Pull Request لمقال، مرئيات، أو بيانات (اتبع إرشادات المساهمة داخل المستودع)
- المشاركة في ورش العمل أو مجموعات العمل التي ربما نعلن عنها عبر الموقع

## تشغيل وتحرير محليًا
1. استنخِب المستودع:
   ```
   git clone https://github.com/acfs-eg/acfs-eg.github.io.git
   ```
2. أنشئ فرعًا للتعديلات:
   ```
   git checkout -b feature/add-content
   ```
3. عدّل أو أضف ملفات Markdown/HTML أو موارد، ثم:
   ```
   git add .
   git commit -m "Add/Update content for the foresight anthropology site"
   git push origin feature/add-content
   ```
4. قدّم Pull Request إلى الفرع الرئيسي للمراجعة والدمج.
